# СуммаНДС
- Извлеки числовое значение суммы НДС, указанное рядом со ставкой или текстом «в том числе НДС …».
- Сохрани формат с пробелами между группами тысяч и используй точку для отделения копеек.
- Если сумма прописана словами и цифрами, ориентируйся на цифры; пример: «… НДС 20% - 202 602 (…) рубля 67 копеек» ⇒ `202 602.67`.
