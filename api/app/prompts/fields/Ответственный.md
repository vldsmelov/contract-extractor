# Ответственный
- Ищи представителя покупателя (Организации), отвечающего за взаимодействие по поставке и приёмке товара.
- Это не тот же человек, который подписывает документ. Ответсвенный указан только в одном разделе: УСЛОВИЯ ПОСТАВКИ, ПОРЯДОК ПРИЕМКИ ТОВАРА
- Часто идет указание следующим образом: "от Покупателя", а далее должность, ФИО, телефон, почта.
- Ответсвенным не может быть А.С. Гурьев!!!
- Если перечислены должность, телефон, email — верни только Ф.И.О. сотрудника. Игнорируй представителей поставщика.
