# Сумма
- Извлеки финальную денежную сумму договора (чаще всего формулировка «Общая сумма договора составляет …»).
- Используй числовое значение из текста, заменяя запятую на точку для отделения копеек и сохраняя пробелы между группами тысяч.
- Если рядом указаны НДС и сумма прописью, выбери величину, которая относится к полной стоимости договора.
- Сфокусируйся на конце документа (последние ~1000 символов): для примера с фразой «Общая сумма Договора составляет 1 215 616 … рублей 00 копеек» ответ должен быть `1 215 616.00`.
