# СпособОплаты
- Найди разделы об условиях расчётов («Порядок оплаты», «Условия оплаты», «Расчёты сторон» и т.п.).
- Определи модель оплаты и выбери формулировку: «30% аванс/ 70% постоплата», «100% аванс», «100% по факту» (после поставки в течении какого-то времени также считается оплатой по факту) или «иное».
- Иное подразумевает другое соотношение аванса и постоплаты (например, "иное — оплата 50% аванс / 50% постоплата").
