# СуммаПрописью
- Найди текстовое написание общей суммы договора, обычно заключённое в круглые скобки сразу после числового значения.
- Сохрани оригинальную формулировку целиком, включая валюту и копейки («Один миллион … рублей 00 копеек»).
- Если встречается несколько вариантов, выбери тот, что относится к полной сумме договора и расположен ближе к концовке документа.
