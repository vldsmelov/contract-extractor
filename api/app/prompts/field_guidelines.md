# Общие рекомендации по поиску данных
- Проанализируй шапку, таблицы реквизитов и подписи сторон (особенно блоки «ПОДПИСИ СТОРОН» в конце документа) прежде чем переходить к разделам приложений.
- Фокусируйся на числовых значениях, датах и статусах рядом с ключевыми словами ("договор", "контрагент", "НДС", "оплата").
- Если сведения отсутствуют, возвращай безопасное значение в соответствии с типом поля.
- Проверяй как русский, так и английский текст — документы могут содержать двуязычные блоки.
- При обработке сторон договора сначала сопоставь роли ("Покупатель", "Заказчик", "Поставщик", "Исполнитель" и т.д.), затем переноси точные юридические наименования без добавления ролей в текст значений.

Персональные инструкции по полям находятся в отдельных файлах внутри каталога `prompts/fields` и объединяются автоматически.
